---

## 📌 Problem Statement (Hackathon)

- ❌ Users waste time switching between 5-6 different apps
- ❌ Payment & address details repeatedly enter karna padta hai
- ❌ Service providers struggle with visibility & discoverability
- ❌ No proper comparison system for price & quality
- ❌ Emergency services ka dedicated platform nahi hai

## ✅ Our Solution

| Problem | Solution |
|---------|----------|
| Multiple apps | ✅ **Unified Dashboard** – sab services ek jagah |
| Repeated data entry | ✅ **Saved profiles** – ek baar details save |
| Provider visibility | ✅ **Analytics Dashboard** – real-time insights |
| No comparison | ✅ **Compare & Decide** – price & ratings side by side |
| Emergency | ✅ **SOS Button** – alerts all nearby providers instantly |

---

## ✨ Key Features

### 🤖 **AI-Powered Assistant**
- Real-time chat support
- Guides user through cab, grocery, repair, healthcare bookings
- Smart keyword detection & responses

### 🔐 **Complete Auth System**
- Signup / Login with localStorage
- User profiles & booking history saved

### 📊 **Provider Dashboard**
- Real-time analytics: total bookings, revenue, unique users
- Growth tracking & popular services insights

### 🚨 **Emergency SOS**
- One-click alerts all nearby service providers
- Fake email/SMS confirmation system

### 🗺️ **Nearby Services Map**
- Shows nearby providers with distance
- Fake map integration for demo

### 📜 **Booking History**
- All past bookings in one place
- Status tracking (Confirmed/Pending)

### 🌓 **Dark / Light Mode**
- Smooth theme toggle
- Eye-friendly UI

### 🔔 **Real-time Notifications**
- Bell icon with badge count
- Dropdown showing all alerts

### 📧 **Fake Email/SMS Confirmation**
- Simulated booking confirmations
- Emergency alert system
